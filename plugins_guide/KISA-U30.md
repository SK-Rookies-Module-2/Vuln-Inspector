U-30: UMASK 설정 관리
• 중요도: 중
• 점검 목적: 파일 생성 시 기본 권한을 제한하여 보안성 확보.
• 보안 위협: UMASK 값이 적절하지 않으면 생성되는 파일에 과도한 권한(쓰기 등)이 부여될 수 있음.
점검 대상 및 판단 기준
• 양호: UMASK 값이 022 이상인 경우 (예: 022, 027, 077).
• 취약: UMASK 값이 022 미만인 경우 (예: 000, 002).
상세 점검 로직 (Scripting Guide)
• 대상 파일: /etc/profile, /etc/bashrc, /etc/csh.login, .profile, .bashrc 등.
• 로직:
    1. 파일 내 umask 문자열 검색.
    2. 설정된 값이 022 이상인지 확인 (마지막 자리가 2 이상, 중간 자리가 2 이상 등).
    3. 일반적으로 022 설정 확인.
